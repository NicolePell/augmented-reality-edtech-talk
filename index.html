The evolution of augmented reality

1968

The first head-mounted display

1974

Videoplace: An 'Artificial Reality' laboratory

1990

'Augmented Reality' the term is coined

1992

Earliest functioning AR system built for the Air Force

1994

'Dancing in cyberspace' - the first AR theatre production

1998

Virtual yellow marker shown in a live NFL game

1999

NASA use a hybrid synthetic vision system to overlay map data to aid navigation

2000

ARToolkit, a software library using video tracking to overlay computer graphics on a video camera

2009

Esquire magazine bring Robert Downey Jr. alive on the cover

2013

Volkswagen provide virtual step-by-step repair assistance

2014

Google announce Google Glass
